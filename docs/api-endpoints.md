# API Endpoints

## HTML API

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Listings

- `GET /api/listings`
- `POST /api/listings`
- `GET /api/listings/:id`
- `PATCH /api/listings/:id`
- `DELETE /api/listings/:id`

### Chatrooms

- `GET /api/chatrooms`
- `POST /api/chatrooms`
- `GET /api/chatrooms/:id`
- `PATCH /api/chatrooms/:id`
- `DELETE /api/chatrooms/:id`
